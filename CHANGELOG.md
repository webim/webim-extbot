# Журнал изменений

## 0.2.1

- Убрано ошибочное предупреждение "Unsupported message kind 'visitor'" в API 2.0
- Изменены ответы бота на неизвестные параметры входящего запроса и неизвестный текст от посетителя
- Добавлена поддержка в debug-логах бота не-ASCII символов, в том числе кириллицы

## 0.2.0

- Возможность установки бота в ОС и запуск бота командой `extbot`
- Единая ссылка на API 2.0 и 1.0 при использовании Webim 10.3+
- Валидация параметров запуска бота
- Более информативные логи и сообщения бота
- Более качественный код за счёт использования black и ruff

## 0.1.0

- Новые клавиатуры и тексты бота для демонстрации разных функций External Bot API
- API 2.0: получение и отправка файлов, перенаправление на оператора и на отдел, закрытие диалога
- API 1.0: возврат диалога в общую очередь

## 0.0.0

- Простейший бот: отправляет кнопки "One", "Two", "Three", "Four" и сообщает, какая кнопка нажата
